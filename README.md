# deno-notes-app-backend
simple backend untuk note app menggunakan [deno@2.6.8](https://deno.com) dan [oak@17.2.0](https://oakserver.org/).

```bash
# clone repo
git clone https://github.com/aelberthcheong/deno-notes-app-backend

# pindah workdir
cd deno-notes-app-backend

# jalankan program
deno task dev
```
